# ✨ 팬 커머스샵 VeryPeri
### 🔎프로젝트 소개
* 현대의 소비 문화에 새로운 풍경을 제시하며, 
젊은 대상층을 위한 새로운 팬 커머스 샵 프로젝트를 소개합니다.

저희 VERY PERI는 젊은 소비층을 위한 
혁신적인 팬 커머스 샵 프로젝트입니다. 
우리의 목표는 단순한 쇼핑 경험을 넘어서, 
여러분이 사랑하는 아티스트와 더 가까워질 수 있는 
기회를 제공하는 것입니다.
